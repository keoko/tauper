# Tauper
Tauper is a multiplayer game implemented in Elixir.

## Goals
  * Learn Elixir/Phoenix
  * Learn Phoenix Presence
  * Learn LiveView
  * Learn how to deploy Elixir apps in fly.io

## Pre-requisites
  * direnv

## deploy the app
#+BEGIN_SRC bash
fly deploy
#+END_SRC

## gather new translations
#+BEGIN_SRC bash
mix gettext.extract --merge
#+END_SRC
